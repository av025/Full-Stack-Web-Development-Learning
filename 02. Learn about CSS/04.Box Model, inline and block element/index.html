Box Model, Inline and Block Element

Intro to Box Model

The CSS Box Model is a fundamental concept that describes how every HTML element is treated as a rectangular box on the webpage. It consists of four layers—content, padding, border, and margin—which together define the total space an element takes up and how it interacts with other elements in the layout.

Different type of Element

In HTML mainly there was 3 type of elements and they play important role to style and place in our webpage layout.

  • Block Level Element

    Block level element by default acquire whole width of our webpage and place next element from new line.

  • Inline Level Element

    Inline Level Element only takes space according to our content and it allow other element to place next or adjacent to each other. Here we have to note that in inline we can't give padding and height to inline element.

  • Inline Block Level Element

    Now this was special property which was provided by css to place our element according to our layout requirement so we can transform our inline element to have property of block level element also.

This was p element example of block level element that takes whole width of our screen or viewport This was span element which was example of Inline element with only takes content space require.

Components of Box Model

  • Content Box : The Content Box where we write our content which have width and height.
  • Padding :Padding was useful property to create spaces between our element's corner and content box so it have some space and clear proper visual.
  • Border :Border was property to give border to our element
  • Margin :Margin was also used to create space but not internally with other elements so our content seen visual clearly.
